
# Progetto Gestione Studente
Questo script Python definisce una classe `Studente` per gestire i dati anagrafici e la carriera accademica.

## Funzionalità principali:
- **Presentazione**: Metodo per visualizzare i dati dello studente.
- **Gestione Voti**: Aggiunta di voti con validazione (accetta solo valori tra 18 e 30).
- **Media dei voti**: Calcolo automatico della media aritmetica con gestione delle liste vuote per evitare errori.
- **Simulazione studio**: Metodo per descrivere le ore dedicate allo studio.

## Esecuzione
Eseguire il file `main.py` per vedere un esempio di utilizzo della classe con due diverse istanze.
